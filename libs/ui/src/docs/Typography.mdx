# Typography

## Font Families

| Token | Value | Usage |
|-------|-------|-------|
| `--font-sans` | `"Inter", ui-sans-serif, system-ui, -apple-system, sans-serif` | All UI text |
| `--font-mono` | `"JetBrains Mono", ui-monospace, "Cascadia Code", monospace` | Code, data |

Font features enabled: `cv02`, `cv03`, `cv04`, `cv11` (Inter's alternate characters).

## Tailwind Text Utilities

| Class | Size | Usage |
|-------|------|-------|
| `text-xs` | 0.75rem / 1rem | Badges, captions, shortcuts |
| `text-sm` | 0.875rem / 1.25rem | Body text, form labels, table cells |
| `text-base` | 1rem / 1.5rem | Default body |
| `text-lg` | 1.125rem / 1.75rem | Section headings |
| `text-xl` | 1.25rem / 1.75rem | Card titles |
| `text-2xl` | 1.5rem / 2rem | Page headings |
| `text-3xl` | 1.875rem / 2.25rem | Hero headings |

## Font Weight Scale

| Class | Weight | Usage |
|-------|--------|-------|
| `font-normal` | 400 | Body text |
| `font-medium` | 500 | Labels, button text, tab triggers |
| `font-semibold` | 600 | Card titles, headings |
| `font-bold` | 700 | Emphasis, stats |

## Usage in Components

```tsx
{/* Heading */}
<h2 className="text-2xl font-semibold tracking-tight">Routes</h2>

{/* Body */}
<p className="text-sm text-muted-foreground">Find your bus route.</p>

{/* Monospace */}
<code className="font-mono text-sm">Route #138</code>
```
